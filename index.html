Game Design

Personal game development projects.

When Night Comes

As a final year undergraduate project, I developed When Night Comes which is an original concept game created using the Unity Game engine. The game investigates four different Artificial Intelligence scripting techniques including Machine Learning, Finite State Machine, Behaviour Tree and the NavMesh component provided by the Unity game engine.

Software: Unity 3D

Tyler's Quest

Small browser game developed with Phaser 2D game framework. Development written in Javascript. Phaser uses both a Canvas and WebGL renderer internally and can automatically swap between them based on browser support.

Software: Phaser Framework

View Project

The Descent

I worked on the The Descent alongside another programmer Mindaugas Navickas for my third year undergraduate project. It is a narrative driven, interactive game. The game was developed using the JavaScript and C# scripting language inside the Unity games-engine. The 3D environment and models were built sourced from different in-engine assets or were custom-made in Adobe Photoshop.

Software: Unity 3D

The Wonderer

Worked on a small WebGL project. The project is low poly and all of the level design is done by placing each object manually. Developed in Unity using C# and the AI uses Behavior Trees to animate and move across the level.

Software: Unity 3D
